一、预装环境
注意！！！   使用该软件需确保电脑安装「nodejs」 和对应的npm包 「anywhere」

①确认环境是否正确安装：
1、打开命令提示符或终端（CMD），输入 node -v 和 npm -v 来验证 Node.js 和 npm（Node.js 的包管理器）是否正确安装。
2、打开命令提示符或终端（CMD），输入 anywhere --help 来验证anywhere是否正确安装。

提示：只要没有出现 
“XXX” 不是内部或外部命令，也不是可运行的程序或批处理文件。
即说明环境安装正常。


②安装nodejs
1、下载 Node.js：访问 Node.js 官网 并下载适合您操作系统的最新稳定版。（安装包下载地址：https://nodejs.org/dist/v20.11.0/node-v20.11.0-x64.msi）
2、安装 Node.js：运行下载的安装程序并遵循指南完成安装。
3、参照 ① 确认是否成功安装。

③安装anywhere
1、确保已正确安装nodejs
2、打开命令提示符或终端（CMD），输入 npm install -g anywhere 并回车。
3、参照 ① 确认是否成功安装。

二、使用说明
1）双击“启动.cmd”即可打开浏览器页面。
2）在左上方房间号输入框输入房间号后，点击连接，等待1.5s后，会在下方房间信息展示连接结果，有时可能出现网络拥堵情况，连接失败则再次点击连接即可，正常第二次就能成功。
3）如果出现用户名带**打码的情况请在抖音网页端登录账号。
4）连接成功后可在右侧查看实时弹幕
5）左下方为抽奖登记控制功能区。输入关键字后点击“开始登记”按钮即开始记录。右下方为登记成功的观众昵称。
6）停止登记后可输入中奖号码高亮显示中间观众的昵称。
7）点击重新开始即清空关键字及列表重新设置抽奖信息。

另：如抽奖过程中连接中断需刷新页面重新走以上流程
